# Метаданные

Вы можете добавить метаданные в начало вашего Markdown-файла. Это блок [YAML](https://yaml.org/), обёрнутый тремя дефисами `---` сверху и снизу, и он используется для определения метаданных.

Например, с помощью [title](/api/config/config-frontmatter#title) можно указать заголовок страницы:

```yaml
---
title: Привет, мир
---
```

Вы также можете обращаться к свойствам, объявленным в блоке метаданных, прямо в теле документа, например:

```markdown
---
title: Привет, мир
---

# {frontmatter.title}
```

Свойства, определённые выше, передаются компоненту как `frontmatter`. Таким образом, итоговый вывод будет:

```html
<h1>Привет, мир</h1>
```

:::tip Совет
Список доступных метаданных смотрите в разделе [Конфигурация метаданных](/api/config/config-frontmatter).
:::
