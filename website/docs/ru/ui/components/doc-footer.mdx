# DocFooter

`DocFooter` отображает область футера в нижней части страниц документа, включая ссылку на редактирование, время последнего обновления и навигацию по предыдущей/следующей странице.

## Использование

Этот компонент автоматически отображается в нижней части страниц документа и обычно не требует ручного использования. Для кастомизации можно его извлечь:

```tsx title="theme/components/DocFooter/index.tsx"
import { EditLink, LastUpdated, PrevNextPage } from '@rspress/core/theme';

export function DocFooter() {
  return (
    <footer className="my-doc-footer">
      <EditLink />
      <LastUpdated />
      <PrevNextPage />
      {/* Добавляем свой контент */}
    </footer>
  );
}
```

```mdx preview
import { DocFooter } from '@theme';

<DocFooter />
```

## Включённые компоненты

`DocFooter` состоит из следующих компонентов:

- [EditLink](/ui/components/edit-link) — Отображает ссылку «Отредактировать страницу»
- [LastUpdated](/ui/components/last-updated) — Отображает время последнего обновления страницы
- [PrevNextPage](/ui/components/prev-next-page) — Отображает навигацию по предыдущей/следующей странице
