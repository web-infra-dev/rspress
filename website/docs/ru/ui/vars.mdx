# CSS-переменные

:::warning 🚧 В процессе доработки

:::

Rspress предоставляет часто используемые CSS-переменные. Переопределение переменных проще и удобнее в поддержке, чем замена встроенных React-компонентов.

Вы можете редактировать и просматривать изменения в реальном времени на этой странице, а затем скопировать результат в `theme/index.css`, чтобы переопределить стили. Вот простой пример переопределения стилей:

import { Tabs, Tab } from '@theme';

<Tabs>
<Tab label="theme/index.tsx">

```tsx
import './index.css';
export * from '@rspress/core/theme-original';
```

</Tab>
<Tab label="theme/index.css">
```css
/* Скопируйте сюда код с CSS-переменными, чтобы переопределить стили */
```
</Tab>
</Tabs>

Ниже перечислены некоторые CSS-переменные, предоставляемые Rspress, и их значения по умолчанию:
