# 总览

## 官方插件

官方插件包括:

- [@rspress/plugin-algolia](./algolia)：基于 [DocSearch](https://docsearch.algolia.com)，将 Rspress 内置的搜索功能替换为 [Algolia](https://www.algolia.com/)。
- [@rspress/plugin-api-docgen](./api-docgen)：[react-docgen-typescript](https://github.com/styleguidist/react-docgen-typescript) 和 [documentation](https://github.com/documentationjs/documentation) 的集成插件，用于自动生成 API 的文档描述。
- [@rspress/plugin-client-redirects](./client-redirects)：支持客户端重定向。
- [@rspress/plugin-llms](./llms)：为 Rspress 站点生成 [llms.txt](https://llmstxt.org/) 相关文件，使大模型可以更好地理解你的文档站。
- [@rspress/plugin-playground](./playground)：支持代码块中的组件预览，并提供实时 Playground。
- [@rspress/plugin-preview](./preview)：支持代码块中的组件预览。
- [@rspress/plugin-rss](./rss)：使用 [feed](https://github.com/jpmonette/feed) 为 Rspress 站点生成 RSS 文件。
- [@rspress/plugin-sitemap](./sitemap)：自动生成用于 SEO 的[站点地图 (sitemap)](https://developers.google.com/search/docs/crawling-indexing/sitemaps/overview)，有利于搜索引擎抓取。
- [@rspress/plugin-twoslash](./twoslash)：集成 [Twoslash](https://github.com/twoslashes/twoslash)，用于自动生成带有类型信息的丰富代码块。
- [@rspress/plugin-typedoc](./typedoc)：[TypeDoc](https://github.com/TypeStrong/typedoc) 集成插件，用于自动生成 TS 模块的 API 文档。
