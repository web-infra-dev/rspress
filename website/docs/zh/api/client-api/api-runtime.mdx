# Runtime API

Rspress 暴露了一组运行时 API，帮助你在构建阶段与浏览器中编排自定义的 UI 与行为。

## React Hooks

每个 Hook 都在 [UI Hooks](/ui/hooks) 下提供了独立的说明：

- [`usePageData`](/ui/hooks/use-page-data)：获取页面元数据（标题、路由、Frontmatter 等）。
- [`useLang`](/ui/hooks/use-lang)：读取当前语言并响应语言切换。
- [`useVersion`](/ui/hooks/use-version)：在多版本文档中获取当前版本。
- [`useDark`](/ui/hooks/use-dark)：判断是否启用暗黑模式。
- [`useI18n`](/ui/hooks/use-i18n)：从配置的多语言来源渲染本地化文案。
- [`路由 Hooks`](/ui/hooks/router-hooks)：使用 Rspress 重导出的 `react-router-dom` Hooks，如 `useLocation` 与 `useNavigate`。

> 更多 `useI18n` 细节可参考[国际化](/guide/default-theme/i18n)。
