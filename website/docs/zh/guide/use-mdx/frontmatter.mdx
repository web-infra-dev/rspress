# Frontmatter

你可以在 Markdown 文件的顶部添加 front matter。它是一个使用 [YAML](https://yaml.org/) 格式定义元数据的块，位于文件顶部的三个连字符 `---` 之间。

例如使用 [title](/api/config/config-frontmatter#title) 来指定页面的标题：

```yaml
---
title: Hello world
---
```

例如，你可以使用这些 headers 为 [Open Graph](https://ogp.me/) 指定自定义元标签。

```yaml
---
head:
  - - meta
    - property: og:url
      content: https://example.com/foo/
  - - meta
    - property: og:image
      content: https://example.com/bar.jpg
# - - [htmlTag]
#   - [attributeName]: [attributeValue]
#     [attributeName]: [attributeValue]
---
```

:::tip
查看 [Front matter 配置](/api/config/config-frontmatter) 了解可用的 front matter 配置。
:::
