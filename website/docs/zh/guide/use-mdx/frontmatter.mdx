# Frontmatter

你可以在 Markdown 文件的顶部添加 front matter。它是一个使用 [YAML](https://yaml.org/) 格式定义元数据的块，位于文件顶部的三个连字符 `---` 之间。

例如使用 [title](/api/config/config-frontmatter#title) 来指定页面的标题：

```yaml
---
title: Hello world
---
```

你还可以在正文中访问 front matter 中定义的属性，例如：

```mdx
---
title: Hello world
---

# {frontmatter.title}
```

上述属性会以 `frontmatter` 形式传递给组件，最终输出为：

```html
<h1>Hello world</h1>
```

:::tip
查看 [Front matter 配置](/api/config/config-frontmatter) 了解可用的 front matter 配置。
:::
