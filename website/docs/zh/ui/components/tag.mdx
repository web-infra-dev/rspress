import { Badge } from '@theme';

# Tag

<Badge text="ejectable" />

Tag 组件用于在侧边栏或导航栏中进行 UI 信息展示。它支持多种格式，包括常用标签、SVG 图标、图片和纯文本。

## 使用方法

通过 [frontmatter](../../guide/use-mdx/frontmatter.mdx) 使用，之后在侧边栏或者导航栏会进行对应展示：

```mdx title="foo.mdx"
---
tag: new
---

# Foo

some text
```

<Tag tag="new" />

同时支持多个标签：

```mdx
---
tag: new, experimental
---
```

<div style={{ display: 'flex', gap: '8px' }}>
  <Tag tag="new, experimental" />
</div>

也可通过 `_meta.json` 使用，例如：

```json title="_meta.json"
{
  "name": "foo",
  "type": "file",
  "tag": "new, experimental"
}
```

:::tip 提示

Tag 组件专门为在侧边栏或者导航栏中展示设计，在 MDX 中使用时，请使用 Badge 组件。

:::

## Tag 种类

为确保 frontmatter 中传值方便，Tag 组件支持以下多种标签类型：

### 常用标签

内置支持了一些常用标签，会配合 `<Badge />` 进行展示，全部常用标签如下：

import { Tag } from '@theme';

| tag            | 对应的 Badge UI 显示       |
| -------------- | -------------------------- |
| `tip`          | <Tag tag="tip" />          |
| `info`         | <Tag tag="info" />         |
| `warning`      | <Tag tag="warning" />      |
| `danger`       | <Tag tag="danger" />       |
| `new`          | <Tag tag="new" />          |
| `experimental` | <Tag tag="experimental" /> |
| `deprecated`   | <Tag tag="deprecated" />   |

Tag 组件是可被 eject 的，当你想扩充常用标签时，可使用 eject 来对 Tag 组件进行修改。

### SVG 字符串

你可以直接传递 SVG 字符串作为标签：

```mdx
---
tag: <svg xmlns="http://www.w3.org/2000/svg" width="20" height="20" viewBox="0 0 24 24"><path fill="currentColor" d="M12 2L2 7l10 5 10-5-10-5z"/></svg>
---
```

### 图片 URL

你可以使用外部 URL 或 data URL 作为标签：

```mdx
---
tag: https://example.com/icon.png
---
```

```mdx
---
tag: data:image/svg+xml;base64,...
---
```

### 纯文本

任何不匹配上述模式的文本都会显示为纯文本 Badge：

```tsx
import { Tag } from '@theme';

<Tag tag="v1.0.0" />
<Tag tag="Beta" />
```

<div style={{ display: 'flex', gap: '8px' }}>
  <Tag tag="v1.0.0" />
  <Tag tag="Beta" />
</div>

## Props

```ts
interface TagProps {
  /**
   * 标签内容。支持：
   * - 常用标签：'tip'、'info'、'warning'、'danger'、'new'、'experimental'、'deprecated'
   * - 多个常用标签：'new, experimental'
   * - SVG 字符串：'<svg>...</svg>'
   * - 图片 URL：'https://...' 或 'data:...'
   * - 纯文本：任意其他字符串
   */
  tag?: string;
}
```
