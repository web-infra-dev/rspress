# Overview

`Overview` 用于页面顶部的概览卡片，常用于快速展示子页面列表或关键信息。

## 用法

```mdx
import { Overview } from '@rspress/core/theme';

<Overview
  items={[
    {
      title: '快速开始',
      link: '/guide/start/introduction',
      description: '安装、初始化项目',
    },
    {
      title: '主题',
      link: '/guide/basic/default-theme',
      description: '主题与布局',
    },
    {
      title: '插件',
      link: '/plugins/overview',
      description: '官方与自定义插件',
    },
  ]}
/>
;
```

### Props

```ts
interface OverviewItem {
  title: string;
  link: string;
  description?: string;
}

interface OverviewProps {
  items: OverviewItem[];
}
```
