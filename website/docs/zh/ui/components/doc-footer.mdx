# DocFooter

`DocFooter` 用于渲染文档页面底部的页脚区域，包含编辑链接、最后更新时间和上一页/下一页导航。

## 用法

该组件会在文档页面底部自动渲染，通常不需要手动使用。如需自定义，可通过 eject 修改：

```tsx title="theme/components/DocFooter/index.tsx"
import { EditLink, LastUpdated, PrevNextPage } from '@rspress/core/theme';

export function DocFooter() {
  return (
    <footer className="my-doc-footer">
      <EditLink />
      <LastUpdated />
      <PrevNextPage />
      {/* 添加自定义内容 */}
    </footer>
  );
}
```

```mdx preview
import { DocFooter } from '@theme';

<DocFooter />
```

## 包含的组件

`DocFooter` 由以下组件组成：

- [EditLink](/ui/components/edit-link) - 显示"编辑此页"链接
- [LastUpdated](/ui/components/last-updated) - 显示页面最后更新时间
- [PrevNextPage](/ui/components/prev-next-page) - 显示上一页/下一页导航
