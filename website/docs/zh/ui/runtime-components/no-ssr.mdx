# NoSSR

`NoSSR` 用于禁用特定子树在服务端渲染，强制只在客户端渲染。

## 用法

```mdx title="index.mdx"
import { NoSSR } from '@rspress/core/runtime';

<NoSSR>
  <ThirdPartyWidget />
</NoSSR>
```

常用于需要避免 SSR，但代码本身仍然可以在 SSG 阶段求值的组件。

如果代码本身依赖浏览器 API，或者需要动态导入浏览器专用依赖，请使用 [`BrowserOnly`](./browser-only)。
