# NoSSR

:::danger 已废弃

`NoSSR` 已废弃，请使用 [`BrowserOnly`](./browser-only)。

:::

`NoSSR` 用于跳过其子树的服务端渲染。

## 用法

```mdx title="index.mdx"
import { NoSSR } from '@rspress/core/runtime';

<NoSSR>
  <ThirdPartyWidget />
</NoSSR>
```

## 另见

- [`BrowserOnly`](./browser-only) — 仅在浏览器 hydration 后渲染内容。
