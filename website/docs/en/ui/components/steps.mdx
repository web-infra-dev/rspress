# Steps

`Steps` renders Markdown content into step-by-step instruction blocks.

## Usage

import { Steps } from '@rspress/core/theme';

<Steps>

### Step 1

Content for step 1.

### Step 2

> Content for step 2.

</Steps>

```mdx title="index.mdx"
import { Steps } from '@rspress/core/theme';

<Steps>

### Step 1

Content for step 1.

### Step 2

> Content for step 2.

</Steps>
```
