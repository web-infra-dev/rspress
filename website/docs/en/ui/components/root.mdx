# Root

`Root` is a wrapper component for the entire application, wrapping all content including the Layout and global UI components.

## Usage

By default, the `Root` component simply renders its children without any additional markup or styling:

```tsx title="theme/components/Root/index.tsx"
import { Root } from '@rspress/core/theme';

function App() {
  return (
    <Root>
      <YourContent />
    </Root>
  );
}
```

## Customizing Root

You can [eject](/guide/advanced/custom-theme.html#eject-component) this component to add global providers, styles, or any other elements that should wrap your entire application:

```tsx title="theme/components/Root/index.tsx"
import type { ReactNode } from 'react';

export interface RootProps {
  children: ReactNode;
}

export function Root({ children }: RootProps) {
  return (
    <YourGlobalProvider>
      <div className="my-custom-root">{children}</div>
    </YourGlobalProvider>
  );
}
```

## Props

### children

- **Type:** `ReactNode`
- **Required:** Yes

The children elements to render, containing the entire application content.

## Use Cases

- Adding global state management providers (e.g., Redux, Zustand)
- Adding theme providers
- Adding global style wrappers
- Adding global error boundaries
- Injecting global scripts or analytics code
