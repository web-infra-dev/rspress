# 容器

你可以使用 `:::` 语法创建自定义容器，并支持自定义标题。

## 示例

:::tip
这是一个 `tip` 类型的 `block`
:::

:::info
这是一个 `info` 类型的 `block`
:::

:::warning
这是一个 `warning` 类型的 `block`
:::

:::danger
这是一个 `danger` 类型的 `block`
:::

::: details
这是一个 `details` 类型的 `block`
:::

:::tip Custom Title
这是一个 `Custom Title` 的 `block`
:::

:::tip{title="Custom Title"}
这是一个 `Custom Title` 的 `block`
:::
