# @rspress/container-syntax

> [Source Code](https://github.com/web-infra-dev/rspress/tree/main/packages/plugin-container-syntax)

Rspress's built-in plugin. It mainly have two functions:

- Parse the container syntax in Markdown/MDX, such as `:::tip`, `:::warning`, and compile them into JSX syntax.
- Add styles to the container block.
